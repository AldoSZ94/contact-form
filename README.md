# ⭐ Contact Form

Proyecto de implementación frontend basado en el reto de Frontend Mentor **"Contact Form"**.  
El objetivo fue construir un formulario de contacto completamente funcional y accesible, aplicando validación de formularios con JavaScript vanilla, manejo del DOM, diseño responsive con enfoque mobile-first y feedback dinámico al usuario mediante mensajes de error y confirmación de envío.

---

## 📌 Descripción

Este proyecto consiste en un formulario de contacto moderno y responsive que permite al usuario ingresar información personal, seleccionar el tipo de consulta, escribir un mensaje y aceptar el consentimiento antes de enviar el formulario.

Se implementó una validación completa tanto a nivel de HTML como de JavaScript, utilizando la Constraint Validation API para asegurar la integridad de los datos y mejorar la experiencia del usuario con retroalimentación inmediata.

Se puso especial énfasis en:

- Estructura semántica HTML
- Diseño responsive con enfoque mobile-first
- Validación de formularios con JavaScript vanilla
- Uso de la Constraint Validation API (`checkValidity`)
- Manejo de eventos del DOM (`submit`, `input`, `change`)
- Validación en tiempo real para inputs y radios
- Control de estado de formulario
- Mensajes de error dinámicos
- Mensaje de éxito al enviar el formulario

## 🎯 Características principales

- 📱 Diseño completamente responsive para mobile, tablet y desktop
- 🎨 UI moderna con Tailwind CSS
- 🧭 Formulario estructurado con campos de nombre, email, mensaje y consentimiento
- ⚡ Validación nativa del navegador con `checkValidity()`
- 🧠 Validación personalizada con JavaScript
- 🔄 Validación en tiempo real para inputs (`input event`)
- 🔘 Validación de grupo de radio buttons (`query-type`)
- ⚠️ Mensajes de error dinámicos por campo
- ✅ Mensaje de éxito al completar el formulario
- 🎯 Control de estado del formulario con lógica condicional
- ♿ Mejora de accesibilidad con etiquetas asociadas (`label` + `for`)
